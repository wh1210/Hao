# 171. Excel Sheet Column Number

```java
class Solution {
    public int titleToNumber(String s) {
        char[] c = s.toCharArray();
        int res = 0;
        for (int i = 0; i < c.length; i++) {
            res += (c[i] - 'A' + 1) * Math.pow(26, c.length - 1 - i);
        }
        return res;
    }
}
```
