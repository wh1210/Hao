# 155. Min Stack

// Tricky solution using one stack.

class MinStack { int min; Stack s; /_\* initialize your data structure here._ / public MinStack\(\) { s = new Stack&lt;&gt;\(\); min = Integer.MAX\_VALUE; }

```text
public void push(int x) {
    if (x <= min) {
        s.push(min);
        min = x;
    }
    s.push(x);
}

public void pop() {
    if (s.pop() == min) {
        min = s.pop();
    }
}

public int top() {
    return s.peek();
}

public int getMin() {
    return min;
}
```
