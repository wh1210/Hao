# 019. Remove Nth Node From End of List

/\*\*

* Definition for singly-linked list.
* public class ListNode {
* int val;
* ListNode next;
* ListNode\(int x\) { val = x; }
* }

  \*/

// One pass class Solution { public ListNode removeNthFromEnd\(ListNode head, int n\) { ListNode prehead = new ListNode\(0\); prehead.next = head; ListNode low = prehead; ListNode high = prehead;

```text
    for (int i = 0; i < n; i++) {
        high = high.next;
    }

    while (high.next != null) {
        high = high.next;
        low = low.next;
    }

    low.next = low.next.next;

    return prehead.next;
}
```
