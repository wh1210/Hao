# 487. Max Consecutive Ones II

//---------------------------------------------------------------- // Not deal with the follow-up class Solution { public int findMaxConsecutiveOnes\(int\[\] nums\) { int numZeros = 0; int res = 0; for \(int lo = 0, hi = 0; hi &lt; nums.length; hi++\) { if \(nums\[hi\] == 0\) { numZeros++; } while \(numZeros &gt; 1\) { if \(nums\[lo++\] == 0\) { numZeros--; } } res = Math.max\(res, hi - lo + 1\); } return res; } }
