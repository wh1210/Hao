# 266. Palindrome Permutation

```java
class Solution {
    public boolean canPermutePalindrome(String s) {
        int[] res = new int[256];
        for (int i = 0; i < s.length(); i++) {
            res[s.charAt(i)]++;
        }
        
        int result = 0;
        for (int temp : res) {
            if (temp % 2 != 0) {
                result++;
                if (result == 2) {
                    return false;
                }
            } 
        }
        
        return true;
    }
}
```
