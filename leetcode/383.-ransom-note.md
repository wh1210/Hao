# 383. Ransom Note

class Solution { public boolean canConstruct\(String ransomNote, String magazine\) { int\[\] rans = new int\[26\]; int\[\] maga = new int\[26\];

```text
    for (char c : ransomNote.toCharArray()) {
        rans[c - 'a']++;
    }

    for (char c : magazine.toCharArray()) {
        maga[c - 'a']++;
    }

    for (int i = 0; i < 26; i++) {
        if (rans[i] > maga[i]) {
            return false;
        }
    }

    return true;
}
```

}
